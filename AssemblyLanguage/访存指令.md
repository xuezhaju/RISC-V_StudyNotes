# 2.2 访存指令

## 寄存器与存储器的差别
- 32个32位寄存器组只有128个字节的容量
- DRAM存储器的容量至少有数GB
- 寄存器的访问速度是DRAM的100倍到500倍

### 如何将数据从存储器装载到寄存器中(Load from Memory to Register)
- lw: Load Word

代码对应：
```C 
int A[100];
g = h + A[3];
```

```Assembly
lw x10,12(x15)     # Reg x10 gets A[3]
add x11, x12, x10  # g = h + A[3]
```
### 如何将数据从寄存器中保存到存储器中(Store from Register to Memory)
- sw: Store Word

```C 
int A[100];
A[10] = h + A[3];
```

```Assembly
lw x10, 12(x15)
add x10, x12, x10
sw x10, 40(x15)
```
