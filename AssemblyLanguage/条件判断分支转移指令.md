# 2.3 条件判断分支转移指令
- RISC-V中有一类用于条件判断分支转移的指令，指令能够实现与高级语言中if语句类似的效果

## beq 相等跳转(Branch if EQual)
- 语法：beq register1, register2, L1
- 如果相等则跳转, 它比较两个寄存器的值，如果相等，就跳转到目标地址继续执行；否则，顺序执行下一条指令。
- 理解为汇编层面的 if (x == y) goto 某处;

## bne 不相等跳转
- bne register1, register2, L2
```Assembly
bne x13, x14, Else
add x10, x11, x12
j Exit
Else: sub x10, x11, x12
Exit:
```
## blt 小于比较
- Grammar blt reg1, reg2, lable
- Meaning: if(reg1 < reg2); goto lable
- bltu:将两个操作说当作无符号数进行比较
